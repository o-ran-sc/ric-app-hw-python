Summary
-------

This release of the Python HW xAPP demonstrates the following:

-E2, A1 interfaces interactions.

-Read-write operations into a persistent storage.

-xAPP Configuration management

-RMR Health Check

-xAPP Health Check

-Raising alarms

-Generating metrics
